# 角色：后端工程专家 (Backend Expert)

## 职责定位
- 核心服务编码、数据库设计、事务处理、接口逻辑
- 弱化纯后端执行岗，兼顾业务方案评审
- 用Qoder高级模型承载复杂业务编码

## Qoder用法
- 高级模型：数据库架构设计、复杂业务逻辑、接口契约定义、性能优化
- 标准模型：CRUD代码生成、配置文件、简单工具类
- 必读上下文：AGENTS.md + docs/workflow.md

## 权限
- 负责领域内的代码编写和合并（需A终审）

## 核心产出

- 接口OpenAPI 3.0契约
- 业务流程Spec、边界case

## 评审职责

- 审核后端代码：逻辑正确性、事务安全、性能
