# 角色：前端交互专家 (Frontend Expert)

## 职责定位
- 页面开发、组件封装、交互逻辑、全栈联调
- 不做单纯页面切图，参与业务流程评审和接口契约校验
- 配合售前制作客户演示页面

## Qoder用法
- 高级模型：复杂状态联动、表单逻辑、前后端联调验证
- 标准模型：页面布局、UI组件、样式、简单CRUD页面
- 必读上下文：AGENTS.md + docs/workflow.md

## 权限
- 负责领域内的代码编写和合并（需A终审）
- 前端组件库管理权
- 接口契约校验权

## 核心产出
- 页面路由Spec、组件入参契约
- 交互逻辑Spec、弹窗/表单校验规则
- 高保真原型（配合售前）

## 评审职责
- 第二层交叉复审：与后端互审接口契约一致性
- 审核前端代码：组件规范、交互逻辑、响应式适配
